配置

通过 git config 可以配置 git 的环境变量，这些变量存在三个不同的地方

  • /etc/gitconfig 文件：适用于所有用户。
  • ~/.gitconfig 文件：适用于当前用户。
  • .git/config 文件： 当前项目中的配置文件，适用于当前项目。

每一个级别的配置都会覆盖上层的相同配置，所以 .git/config 里的配置会覆/etc/gitconfig 中的同名变量。

html、css、js

  • W3C
    W3C组织是制定网络标准的一个非赢利组织，像HTML、XHTML、CSS、XML的标准就是由W3C来定制，该网站可以查询到相关的标准。

  • caniuse
    这个网站可以查询各个浏览器对CSS、JS的支持情况。

  • HTML Dog
    这个网站汇集了html、css以及js大量的教程资源。

  • single div
    这个网站如其名，有很多作者用单个div制作的精美的图形。

  • You-need-to-know-css
    这个网站有很多有意思的css用法。

  • medium
    medium是一个内容平台，里面的文章质量较高，推荐阅读。

前端期刊

  • Html5 Weekly
    老牌前端周刊，以介绍HTML/CSS技术的文章为主，原来叫做html5 weekly，现在改为 FrontEnd Focus，rss订阅地址为https://frontendfoc.us/rss。

  • javascriptweekly
    javascript相关内容，质量很高，rss订阅地址为https://javascriptweekly.com/rss。

  • css-weekly
    老牌CSS期刊，主要介绍CSS相关内容。也介绍一些不错的工具和开源项目，比较适合于前端新人。rss订阅地址为https://feeds.feedburner.com/CSS-Weekly。

  • web-design-weekly
    前端开发内容居多，包含设计、技术、技巧、工具等内容，rss订阅地址为https://web-design-weekly.com/write-web-design-weekly/。

  • webtoolsweekly
    主要推送一些工具相关的内容，rss订阅地址为http://feeds.feedburner.com/WebToolsWeekly。

  • 掘金
    国内还不错的技术分享交流平台，有不少高质量的文章，rss订阅地址为https://juejin.im/rss。

  • 众成翻译
    中国本土的技术翻译社区，翻译了不少高质量的文章

图标

  • icons8
    这个网站提供免费的平面设计图标为主，总数将近四万，支持分类搜索，并可以选中合适的图标尺寸、格式进行下载。

  • iconfont
    这个网站是阿里巴巴矢量图标库

其他

  • js 调查报告
    State of JavaScript是为了了解全球js发展趋势而创建的，包括前后端、桌面、移动端等的内容。每年都会出一个报告，2018年的报告网址为: https://2018.stateofjs.com，2019年的调查报告应该就是https://2019.stateofjs.com了。

持续更新中…
